# Pollen Theme for Obsidian

A scholarly dark cyan and warm sepia-gray theme built for focus, reading, and deep work.

## Features
- **Embedded Typography:** Comes with Gentium Plus (SIL Open Font License) for beautiful, highly readable body text, and Fira Code for crisp monospace headings. Both are base64-embedded directly into the theme (no external downloads or system installations required).
- **Reduced Eye-Strain:** Uses a unified #141414 dark background and muted sepia-gray text to reduce contrast harshness.
- **Dashed Indentation Guides:** Features custom --- dashed guides in the file explorer to easily track folder depth.
- **Light Theme Parity:** Fully supports light mode with carefully adjusted contrast and visibility.
- **Native Font Support:** Fully respects custom fonts chosen in Obsidian's Appearance settings.

## Customization (Style Settings)
This theme is fully compatible with the **Style Settings** community plugin. Once both are installed, go to **Settings → Style Settings → Pollen** to customize:
- Heading colors and fonts
- Body text typography
- Background colors and border opacities
- Indentation guide dash and gap lengths
- Tab close button scale and pin icons
